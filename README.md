# Fabric TPS

### *A mod by*
![Rupyber Studios Logo](https://rupyberstudios.github.io/website/img/rupyber_studios.png)

Adds an equivalent to the '/forge tps' command:
- `/fabric tps` (also `/quilt tps`)
- `/fabric tps fancy` (also `/quilt tps fancy`)
