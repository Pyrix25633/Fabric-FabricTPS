# Minebucks Currency

### *A mod by Rupyber Studios*

This is a Fabric and Quilt Mod that adds an equivalent to `/forge tps` command:
- `/fabric tps` (also `/quilt tps`)
- `/fabric tps fancy` (also `/quilt tps fancy`)
